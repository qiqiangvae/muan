# Mysql 一些命令和sql记录

## Sql

### 字符串转数字

```sql
-- 转整型，无符号用 unsigned
select cast('80484.55' as signed );
select convert('80484.55', signed);
-- 转小数
select cast('80484.55' as decimal(20,2));
select convert('80484.55', decimal(20, 2));
```

convert和cast支持的数据类型有:

- binary：二进制类型；
- char：字符类型；
- date：日期类型；
- time：时间类型；
- datetime：日期时间类型；
- decimal：浮点型；
- signed：整型；
- unsigned：无符号整型。
