# Arthas线上排查经验

## 介绍

Arthas 是 Alibaba 开源的 Java 诊断工具。

[官网地址](https://arthas.gitee.io/index.html)

## 快速安装

```shell
curl -O https://arthas.aliyun.com/arthas-boot.jar
java -jar arthas-boot.jar
```

## 常用命令

### 反编译源码
```shell
jad com.example.test
```
