### zsh 安装配置

安装 **zsh**

```shell
# ubuntu 安装命令
apt install zsh
```

安装 **oh-my-zsh** 

```shell
# 国内访问更快
sh -c "$(curl -fsSL https://gitee.com/mirrors/oh-my-zsh/raw/master/tools/install.sh)"
```
推荐主题 `steeef`或者`ys`。
```shell
vim .zshrc
```
配置主题样式名称
```text
ZSH_THEME="steeef"
```

### brew 安装

推荐这个仓库 <https://gitee.com/cunkai/HomebrewCN>。

一键安装并设置国内镜像源

```shell
/bin/zsh -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

### vim 配置

在`~/.vimrc`中添加如下配置，如果没有该文件则新建文件。

```
# 设置编码格式
set encoding=utf-8
# 设置行号
set number
# 设置高亮
syntax on
# 自动缩进
set autoindent
```
